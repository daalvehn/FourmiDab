But du jeu :
============

Envoyer la fourmis de l'espace dans le trou noir, en l'attirant avec les planètes bonbon
et en la repoussant avec les planètes citron.

Attention aux différents obstacles ! 

Voyager de niveau en niveau jusqu'a atteindre votre but :

Parviendrez-vous à atteindre la Marmelanète ???


Touches :
=========

 - Touche E (ou clic gauche souris sur le slot) pour sélectionner le slot 1
 - Touche R (ou clic gauche souris sur le slot) pour sélectionner le slot 2

 - Slot 1 = planète bonbon (attire la fourmis)
 - Slot 2 = planète citron (repousse la fourmis) 

 - Clic gauche dans la zone de jeu pour poser une planète (celle sélectionnée)
 - Clic droit sur une planète posée pour la récupérer
